easySKOS: Introduction

What is easySKOS?

easySKOS is a free web service that converts CSV or XLSX spreedsheet files into SKOS-based thesaurus, using underneath the power engine of RMLmapper, that is a a Java-based library that executes RML rules to generate Linked Data. SKOS was chosen because it is a common data model for sharing and linking knowledge organization systems, such as thesauri, taxonomies or classifications schemes, across the Web.

What can be done with it?

easySKOS is still under development, but it provides the following features:
  • Transform your CSV data into SKOS-based thesaurus following the templates provided.
  • Transform your XLSX spreedsheet (Excel) data into SKOS-based thesaurus following the templates provided.
  • Validate your SKOS-based thesaurus (In progress).

For whom has it been built?

easySKOS is an easy-to-use tool whom targe public is everyone who is will to transform her tabular data in SKOS thesaurus following the guidelines of Linked Data.

Who build it?

easySKOS has been built by the undergraduates students from the Polytechnique University of Madrid, with the valuable help of the Ontology Engineering Group.
